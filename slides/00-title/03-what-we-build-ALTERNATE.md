# What You'll Build Today

## FIFA Atlanta Navigator

<div class="mt-4">

### Features You'll Implement

✅ **Google Maps Integration**
- Live traffic layer (green/yellow/red roads)
- Interactive pan, zoom, rotate
- Centered on Mercedes-Benz Stadium

✅ **Real-Time MARTA Transit**
- Blue markers: Bus locations
- Red markers: Train stations/routes
- Auto-refresh every 30 seconds

✅ **FIFA Match Schedule**
- Upcoming matches sidebar
- Event details with times
- Click to center map on venue

✅ **Full Internationalization**
- EN, ES, DE, KO, FR supported
- Language switcher in header
- All UI text translates
